# Loan Default Risk Prediction

An end-to-end machine learning project to predict the likelihood of loan default using classification models (Logistic Regression and XGBoost).

## 📌 Project Overview
This project analyzes loan applicant data to predict the risk of default. It includes data preprocessing, EDA, feature engineering, model training, and a deployment-ready app.

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **Libraries**: pandas, numpy, scikit-learn, matplotlib, seaborn, Logistic Regression and xgboost
- **App**: Streamlit
- **Notebooks**: Jupyter
